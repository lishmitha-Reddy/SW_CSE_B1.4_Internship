CSS Colors : set the color of text and background using CSS

1. Named Colors: red/blue/green 2. HEX colors : Colors written in hexadecimal digits. 6 digits: red_ _ green _ _ blue _ _ 3. RGD &RGBA colors : Colors written in RGB values. 3 values: red, green, blue. RGBA adds an alpha channel for transparency. 4. HSL & HSLA colors : Colors defined by hue, saturation, and lightness. HSLA adds an alpha channel for transparency.
